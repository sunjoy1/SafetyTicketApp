# 安全管理作业票系统 - Android 应用

## 项目概述

基于 GB 30871-2022 标准开发的安全管理作业票 Android 应用，支持高处作业、有限空间作业、动火作业等特种作业票的数字化办理、签字、审批、拍照、存档与打印。

## 核心功能

| 功能 | 说明 |
|------|------|
| 作业票创建 | 依据国家标准模板，填写高处/有限空间/动火作业票 |
| 电子签字 | 支持触屏手写签字，保存为图片并嵌入作业票 |
| 审批流程 | 多级审批：作业负责人→车间→安全部门→厂领导→完工验收 |
| 现场拍照 | 调用系统相机，拍摄作业现场照片存档 |
| 历史存档 | 本地 SQLite 数据库存储，支持搜索查询 |
| 打印输出 | 生成标准 PDF 文档，签字自动填入对应区域，支持蓝牙/网络打印 |

## 作业票类型

### 1. 高处（登高）作业票
- 编号规则：DH + 年月日 + 序号
- 支持4级高度分级（一级2-5m / 二级5-15m / 三级15-30m / 四级>30m）
- 14项标准安全措施确认

### 2. 有限空间作业票
- 编号规则：YX + 年月日 + 序号
- 13项标准安全措施确认
- 气体检测、通风、隔离等专项要求

### 3. 动火作业票
- 编号规则：DHF + 年月日 + 序号
- 支持三级分级（特级 / 一级 / 二级）
- 16项标准安全措施确认
- 包含动火前验票环节

## 技术架构

- **语言**：Java
- **最低 SDK**：Android 7.0 (API 24)
- **目标 SDK**：Android 14 (API 34)
- **数据库**：SQLite（5张表：tickets / ticket_fields / safety_measures / signatures / photos）
- **签字**：自定义 Canvas 手写 View
- **打印**：Android PrintManager + PDF 生成
- **拍照**：Intent 调用系统相机 + FileProvider

## 使用说明

### 创建作业票
1. 打开应用，点击右下角「+」按钮
2. 选择作业票类型（高处/有限空间/动火）
3. 填写基本信息：申请单位、作业地点、作业内容、风险辨识、作业时间
4. 选择作业级别（如高处作业选择高度级别）
5. 逐条确认安全措施（是/否/不涉及）并填写确认人
6. 点击「保存草稿」或「提交审批」

### 电子签字
1. 在创建/编辑页面点击「签字」按钮
2. 选择签字类型（申请人/监护人/作业人）
3. 在签字区域手写签名
4. 输入签字人姓名，点击「保存签字」

### 审批流程
1. 提交审批后，作业票进入「待审批」状态
2. 在详情页点击「审批」按钮
3. 按顺序进行：作业负责人 → 所在单位（车间） → 安全管理部门 → 厂领导/总工程师
4. 每一级必须手写签字并填写审批意见
5. 全部审批通过后，作业票变为「已通过」状态

### 现场拍照
1. 在创建/编辑/详情页点击「现场拍照」或「查看/添加现场照片」
2. 输入照片描述（如：作业现场全景）
3. 点击「拍摄现场照片」，调用系统相机拍摄
4. 照片自动关联到当前作业票
5. 长按照片可删除

### 打印作业票
1. 在作业票详情页点击「打印」按钮
2. 系统生成标准 PDF 文档
3. 签字图片自动嵌入对应区域
4. 选择打印机（蓝牙/网络/系统打印）输出

### 搜索与存档
1. 首页顶部搜索框可按编号、单位、地点、内容搜索
2. 所有作业票数据保存在本地数据库
3. 长按作业票列表项可删除
4. 支持按作业票类型筛选（可在代码中扩展）

## 开发环境要求

- Android Studio Hedgehog (2023.1.1) 或更高版本
- JDK 17
- Gradle 8.2
- Android SDK 34

## 构建与运行

1. 用 Android Studio 打开 `SafetyTicketApp` 目录
2. 等待 Gradle 同步完成
3. 连接 Android 设备或启动模拟器
4. 点击「Run」按钮编译安装

## 注意事项

1. 首次运行需要授予相机权限和存储权限
2. 打印功能需要 Android 设备支持打印服务（可通过系统设置连接打印机）
3. 签字数据以 PNG 格式存储在数据库中，确保签字清晰
4. 现场照片存储在应用外部私有目录，卸载应用时可能保留
5. 建议在正式使用前根据企业实际流程调整审批环节

## 安全规范依据

本应用作业票模板依据以下标准设计：
- GB 30871-2022《危险化学品企业特殊作业安全规范》
- 高处作业票参考 GB 30871-2022 附录 A
- 有限空间作业票参考 GB 30871-2022 附录 B
- 动火作业票参考 GB 30871-2022 附录 C
